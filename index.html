Manual de Instalação do Aplicativo Android - ebook prenatal

Passo 1: Acesse o Link de Instalação

Abra o Google Chrome no seu dispositivo Android e acesse o link de instalação do aplicativo ebook prenatal.

Passo 2: abra o link

Na página de instalação, clique no botão abaixo para acessar o drive do ebook prenatal:

Baixar APK do ebook prenatal da UNEB

Passo 3: Faça o download

Na página de Download do drive, clique no botão "download" para baixar o arquivo APK do aplicativo ebook prenatal:

Passo 4: Permitir Instalação de Fontes Desconhecidas

Ao tentar instalar, o sistema pode solicitar que você habilite a instalação de aplicativos de fontes desconhecidas. Siga as instruções na tela para permitir esta opção:

  • Vá até Configurações > Segurança > Fontes desconhecidas.
  • Ative a opção para permitir instalações pelo Chrome.

Passo 5: Concluir a Instalação

Após a permissão, volte ao Chrome e toque em Instalar para concluir a instalação do aplicativo ebook prenatal.

Passo 6: Habilitar Permissões

Ao abrir o aplicativo pela primeira vez, ele pode solicitar permissões para acessar certas funcionalidades do dispositivo, como armazenamento ou câmera.

Conceda essas permissões para que o aplicativo ebook prenatal funcione corretamente.

Nota: Caso tenha dificuldades durante a instalação, verifique se o dispositivo está com espaço de armazenamento suficiente e se a conexão com a internet está estável.
Atenção: Se precisar de assistência, entre em contato com um técnico da UNEB (Universidade Federal da Bahia) para obter suporte na instalação.
